## 🎮 Hero Sky

Jogo 2D criado utilizando Java Swing, orientação a objeto.

### Funcionalidades

- Java nativo sem engines externas
- Sistema de movimentação com teclado (setas)
- Disparo de projéteis com detecção de colisão
- Inimigos com IA simples e spawn dinâmico
- Animação de explosões com sprite sheet
- Cenário com parallax scrolling (céu, montanhas, tijolos)
- Sistema de pontuação e barra de vida
- Loop principal do jogo rodando em Thread separada (~60 FPS)
